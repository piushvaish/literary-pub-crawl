Getting Started with Our Audio Guide

Use any modern web browser like Chrome or Firefox to access our app. When prompted, allow location access so we can guide you accurately. Look for our app's pin on the map - it'll show you exactly where to go.

Found the spot? Great! Tap the pin to reveal a link to your audio guide. The guide is hosted on Google Drive, so you might need to sign in if you're not already logged in.

Once you've opened the file, you're all set to start listening. Feel free to adjust the playback speed to whatever feels comfortable - whether you prefer a slower pace or want to zip through. And don't worry if you miss something - you can always rewind and listen again.
